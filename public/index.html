GitHub技术速递生成器

基于 DeepSeek 的 GitHub 技术领域热点项目推荐

如何使用

1. 输入技术领域

在技术领域输入框中输入您感兴趣的技术关键词，例如：人工智能、前端开发、区块链等。支持多个关键词，用逗号分隔。

2. 自定义提示词（可选）

点击"显示提示词"按钮，输入自定义提示词，例如："更关注工程实践和性能优化"或"更偏向初学者"。

3. 选择日期范围

选择您想要查看的时间范围，可以选择最近7天、15天、30天，或自定义日期范围。

4. 生成技术速递

点击"生成技术速递"按钮，系统将为您生成该技术领域的GitHub热点项目推荐报告。

5. 查看结果

生成完成后，您可以查看详细的技术速递报告，包括项目概述、核心功能、技术栈等信息。

6. 下载报告

点击"下载报告"按钮，将报告保存为HTML文件，方便离线查看和分享。

7. 查看历史记录

点击右上角的"历史记录"按钮，查看您之前生成的技术速递报告。
